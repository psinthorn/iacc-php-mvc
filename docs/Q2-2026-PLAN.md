## Risk Assessment

| Risk | Impact | Mitigation |
|------|--------|------------|
| BOT API unavailable/rate limited | Medium | Cache rates, manual entry fallback |
| EMVCo QR compatibility across banks | High | Test with SCB, KBank, BBL, KTB apps |
| Tax calculation accuracy | Critical | Cross-reference with actual RD filings |
| Multi-currency rounding errors | Medium | Use DECIMAL(16,6) for rates, round at display |
| PromptPay auto-confirm requires bank API | Low | Manual confirm is default, auto is future |
